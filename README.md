# Tabata Timer

A simple, local-first Tabata interval timer that runs entirely in your browser — no install, no server.

## Features

- **4 configurable phases**: Get Ready, Work, Rest, Round Break
- **Named exercises** — add as many as you want per round
- **Voice announcements** via the browser's built-in speech engine
- **Audio countdown beeps** for the last 3 seconds of each phase
- **Color-coded display**: each phase has its own color so you can glance and know what's happening
- **Pause / Resume / Reset** controls

## How to use

1. Open `index.html` in any modern browser (Chrome, Edge, Firefox, Safari)
2. Set your phase durations and number of rounds
3. Add your exercises
4. Press **Start Session**

## Default Tabata settings

| Phase | Default |
|-------|---------|
| Get Ready | 10 s |
| Work | 20 s |
| Rest | 10 s |
| Round Break | 60 s |
| Rounds | 8 |

## Requirements

- A modern browser with Web Speech API support (Chrome / Edge recommended for best voice quality)
- No internet connection required after download
